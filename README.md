# BLStationaryPlaceholderTextField
BLStationaryPlaceholderTextField 一个简单的带有固定左侧文字占位的UITextField,支持xib/storyboard的可视化配置。
注意：若开起showLeftPlaceHolderView，显示填入placeholder text才会有显示，显示文字即为placeholder文字。
